# My-First-Python-Programs
This repository contains my first Python program. It demonstrates basic programming concepts such as variables, input output, and simple logic. This project marks the beginning of my journey in Python programming.

## 🛠️ Problems Covered

| # | Problem              | Description                    |
| - | -------------------- | ------------------------------ |
| 1 | ➕ Sum of Two Numbers | Take input and print their sum |
| 2 | 🔢 Odd or Even       | Check if number is even or odd |
| 3 | 📊 Factorial         | Compute factorial using loop   |
| 4 | 🔁 Fibonacci         | Generate first `n` numbers     |
| 5 | 🔄 Reverse String    | Reverse user input string      |
| 6 | 🔍 Palindrome        | Check if string is palindrome  |
| 7 | 📅 Leap Year         | Check leap year condition      |
| 8 | 💡 Armstrong Number  | Check Armstrong number         |
